## 구현 기능
### 기능1
상세 페이지 이미지 슬라이더<br/>
사용 라이브러리: react-slick
<br/>
### 기능2
상품별 상세페이지 이동<br/>
사용 라이브러리: react-router-dom 의 {Link} import
<br/>
### 기능3
매너온도별 아이콘 및 막대바(색, 길이) 변경
